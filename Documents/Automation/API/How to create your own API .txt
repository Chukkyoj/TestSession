How to create your own API 

2 types of API 

1. Local
2. Remote

Step 1 - Download and install NOde js

Check if node js is installed via CMD
NODE --VERSION
Check if npm is installed
npm -- version

Step 2 - Install json server via cmd  by using 
npm  install -g json-server

Step 3 - Write a javascript code in an editor 

Step 4 - Save as a Json file

Step 5 - Import or input to Postman all parameters needed 

Step 6 - Leave the Json server CMD running 

Step 7 - Enter the json-server filename.json command on cmd(Ensure you are on the location where your json file is saved



HOW TO FIND JSON PATHS

Step 1 - Launch jsonpathfinder (It will show the individual paths)

Step 2 - Launch jsonpath.com (It will show the response from the path)

HOW TO FIND JSON SCHEMA

Step 1 - Launch your browser

Step 2 - Search for  free json schema online

Step 3 - Copy the response and paste on the schema generator

Json Schema checks if the data types are what is given in the response

Things to check in API tests

1. Cookies
2. Headers
3. Status codes
4. Response time
5. Response body

5 Scope of creating variables

1. Global
2. Environment
3. Collection
4. Local
5. Data 

Setting variables from the pre-request script

pm.global.set("Key","Value")
pm.environment.set("Key","Value")
pm.collectionvariables.set("Key","Value")
pm.variables.set("Key","Value")

Removing pre-request scripts

pm.global.unset("Key")
pm.environment.unset("Key")
pm.collectionvariables.unset("Key")
pm.variables.unset("Key")
